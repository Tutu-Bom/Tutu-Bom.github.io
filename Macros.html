Отчет по макросам

Макрос-это записанная последовательность команд, которую затем выполняет Excel.

Сделаем макрос печати:
Sub печать()
Ctrl+P
ActiveCell.FormulaR1C1 = "academia-excel.ru"
Range("P6").Select
End Sub

Sub замена()
Сочетание клавиш: Ctrl+c
Cells.Replace What:="akademia-excel.ru", Replacement:="school", LookAt:= _ xlPart, SearchOrder:=xlByRows, MatchCase:=False, SearchFormat:=False, _ ReplaceFormat:=False
End Sub

Макрос замены шрифта
Sub заменашрифта()
Ctrl+q
Cells.Select
Range("K13").Activate
Selection.Font.Italic = True
Range("M11").Select
End Sub
